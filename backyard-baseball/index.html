Backyard Baseball — Free Baseball Game for Kids

A free browser baseball game at Dad Arcade. Play a full 3-inning game against a CPU team or swing for the fences in Home Run Derby.

About This Game ▾

Full Game mode features 3 innings of batting, pitching with three pitch types, fielding animations, baserunners, and run scoring against one of 10 random CPU teams. Home Run Derby mode offers timing-based hitting with slow-motion replays, fireworks, and a distance leaderboard. Three difficulty levels from kid-friendly to challenging.
